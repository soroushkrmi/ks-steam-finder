# ks-steam-finder
A modular WordPress plugin integrating Steam store scraping (DLCs/Bundles) with WooCommerce custom cart metadata and pricing.
# 🎮 KS Steam Finder (Modular WooCommerce Integration)

یک افزونه پیشرفته، مدرن و کاملاً ماژولار برای وردپرس و ووکامرس که به کاربران اجازه می‌دهد بازی‌ها، محتواهای اضافه (DLC) و باندل‌های پلتفرم Steam را جستجو کرده و با قیمت‌گذاری داینامیک بر اساس نرخ ارز، مستقیماً به سبد خرید سایت اضافه کنند.

---

## ✨ ویژگی‌های کلیدی (Key Features)

* **اسکرپر هوشمند با DOMDocument:** استخراج سریع و دقیق اطلاعات دیتای استیم (شامل DLCها و باندل‌ها) بدون وابستگی به فیلترهای شکننده Regex.
* **پشتیبانی از ریجن‌های چندگانه:** قابلیت پردازش و استخراج قیمت‌ها بر اساس ریجن‌های محبوب (ترکیه 🇹🇷، آرژانتین 🇦🇷، اوکراین 🇺🇦، هند 🇮🇳).
* **سیستم ضد اسپم (Rate Limiting):** حفاظت از منابع سرور و جلوگیری از درخواست‌های تکراری با بهره‌گیری از Transient API وردپرس بر اساس IP کاربر.
* **محاسبه داینامیک قیمت و سود:** تبدیل خودکار ارزهای جهانی به تومان بر اساس نرخ ارزهای تعریف شده در دیتابیس (`ks_rate_usd` و غیره) به همراه احتساب درصد سود اختصاصی سایت (`ks_profit`).
* **یکپارچگی عمیق با سبد خرید ووکامرس:**
    * ذخیره مطمئن اطلاعات حساس اکانت کاربر (یوزرنیم، پسورد، ریجن، وضعیت استیم‌گارد) در Session سبد خرید.
    * تغییر خودکار تصویر شاخص محصول مجازی در سبد خرید به پوستر اصلی بازی دریافت شده از استیم.
    * نمایش ساختاریافته اطلاعات اکانت به عنوان متادیتا در فاکتور سفارشات بخش مدیریت ووکامرس.
* **رابط کاربری مدرن (Cyberpunk Dark UI):** طراحی اختصاصی، تاریک و کاملاً ریسپانسیو فرم‌ها و پاپ‌آپ دو مرحله‌ای تایید اطلاعات.

---

## 📂 ساختار ماژولار پروژه (Project Architecture)

پروژه برای توسعه‌پذیری آسان‌تر به ۴ بخش اصلی تفکیک شده است:
1. `ks-finder.php`: هدر اصلی افزونه و راه‌انداز (Bootstrapper) ماژول‌ها.
2. `includes/class-ks-scraper.php`: لایه ارتباطی با API استیم و استخراج‌کننده ساختار HTML.
3. `includes/class-ks-ajax.php`: هندلرهای درخواست‌های ناهمگام (AJAX) برای لود لیست‌ها، پردازش آیتم‌ها و محاسبات مالی.
4. `includes/class-ks-woocommerce.php`: فیلترها و هوک‌های سبد خرید، سشن‌ها، قیمت‌گذاری و متادیتای سفارشات ووکامرس.
5. `includes/class-ks-shortcode.php`: مدیریت لایه نمایش (UI)، استایل‌های پاپ‌آپ و اسکریپت‌های فرانت‌اند.

## ⚙️ راه‌اندازی و پیش‌نیازها (Requirements & Setup)

* وردپرس نسخه ۵.۰ یا بالاتر
* ووکامرس نسخه ۵.۰ یا بالاتر (محصول مجازی با شناسه `587` باید در سایت موجود باشد)
* فعال بودن توابع cURL روی سرور برای اجرای `wp_remote_get`
